⚙️ Configurações

URLs geradas no Power Automate e SharePoint. Ficam salvas apenas neste aparelho — use "Copiar link de configuração" para configurar o celular do outro operador em 1 toque.

🔑 Grupos de acesso (InControl)

Cada grupo lista os dispositivos (portas/leitores) que ele libera. Fica salvo na planilha central do SharePoint — vale para todos os aparelhos na hora. O CSV de importação leva só o nome do grupo; o vínculo real grupo→dispositivos é configurado dentro do InControl (isto aqui é a referência oficial para os operadores).

👥 Usuários do app

Cadastro central no SharePoint — vale para todos os aparelhos. Cada usuário tem um nível (que já define um conjunto de telas) e você pode personalizar as telas que ele enxerga. As senhas são guardadas com hash (nunca em texto).

Ao editar, deixe a senha em branco para mantê-la.
